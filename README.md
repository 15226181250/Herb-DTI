# Herb-DTI
Herb-DTI is a deep learning based drug target interaction prediction model

## Environmental version requirements
`Python 3.7` packages:
```
- pytorch 1.6+
- pytorch-geometric 1.6+
- numpy 1.19+
- scikit-learn 
- networkx
- gensim
- rdflib
- BioPython
- tqdm
- Groovy (Groovy: 2.4.10 JVM: 1.8.0_121)
- diamond & blastp
```
